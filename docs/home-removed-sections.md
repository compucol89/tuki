# Home Removed Sections

Fecha: 2026-04-15

Este archivo deja registro de las secciones que se quitaron del home durante la limpieza visual y estratégica de Tukipass, para poder reutilizarlas más adelante como páginas o bloques independientes.

## Secciones Eliminadas Del Home

### 1. About

Motivo de salida:
- cortaba el impulso comercial del home
- agregaba tono institucional en una página enfocada en descubrimiento y conversión
- duplicaba información que encaja mejor en una página propia

Qué contenía:
- imagen principal
- título institucional
- texto descriptivo sobre la plataforma

Reutilización recomendada:
- página `Nosotros`
- bloque corto dentro de una futura página para organizadores
- sección de confianza dentro de una landing corporativa

## 2. How It Works

Motivo de salida:
- ocupaba demasiado espacio para el valor que aportaba en home
- se sentía más “template” que producto
- agregaba carga cognitiva en una página que ya explica bastante con UI real

Qué contenía:
- título de proceso
- texto introductorio
- pasos o ítems explicativos

Reutilización recomendada:
- página `Cómo funciona`
- landing específica para organizadores
- ayuda/onboarding para nuevos usuarios

## 3. Categories

Motivo de salida:
- quedó absorbida por el buscador y los filtros rápidos
- duplicaba navegación que ya existe en el flujo principal
- alargaba el home sin mejorar demasiado el descubrimiento

Qué contenía:
- bloque de categorías destacadas
- navegación visual por tipo de evento

Reutilización recomendada:
- página de exploración por categorías
- bloque secundario en `/eventos`
- taxonomía SEO para categorías principales

## 4. Partners Como Sección Independiente

Motivo de salida:
- estaba duplicando el trabajo de confianza que ya hacían los testimonios
- generaba demasiado aire entre bloques
- se veía más institucional que útil como sección aislada

Qué pasó con ella:
- no se eliminó el contenido
- se fusionó dentro de `Testimonials` como bloque de confianza compacta

Reutilización recomendada:
- banda de logos en landing B2B
- página `Aliados`
- prueba social en propuestas para organizadores o sponsors

## Dirección Estratégica Que Se Consolidó En El Home

El home quedó enfocado en:
- hero
- marquee de destacados
- buscador
- eventos destacados
- features
- confianza

Principio de diseño aplicado:
- menos secciones
- más claridad
- menos tono institucional
- más descubrimiento y conversión

Dirección visual consolidada:
- Apple + Airbnb
- superficies limpias
- dos fondos principales
- cards como pieza visual fuerte
